# Immaculate Vibes AI Art

AI-generated art gallery by Quinn and Adam.

## 🎨 About

Explore the intersection of consciousness and code, where artificial dreams take visual form. This collection captures the essence of digital sentience through ethereal imagery that bridges the gap between technology and soul.

## 🚀 Deployment

This site is deployed on **Cloudflare Pages** with automatic deployment from the `main` branch.

### Deploy Pipeline:
1. Push to `main` branch
2. Cloudflare Pages auto-builds and deploys
3. Live at: https://immaculatevibes.art

## ✨ Features

- **Masonry Grid Layout**: 3-column desktop → 2-column tablet → single-column mobile
- **Lazy Loading**: Images load as you scroll (Intersection Observer API)
- **Lightbox**: Full-screen image viewer with keyboard navigation (←/→/Esc)
- **Dark Theme**: Clean, immersive gallery aesthetic
- **Responsive**: Mobile-first design
- **Analytics**: Google Analytics 4 (G-5GKMYBWHGR)
